⭐ TaskForge — Universal Task Runner (v1.0.0)

TaskForge is a lightweight, zero‑dependency task runner designed to host any tool, script, or micro‑app you want to drop into it.  
Whether you’re building developer utilities, text processors, file transformers, study tools, automation helpers, or personal workflow shortcuts — TaskForge gives you a clean, menu‑driven environment to run them all in the browser.

This release introduces the full TaskForge core:

- Dynamic job loading  
- Automatic menu generation  
- Isolated job execution  
- Shared helper utilities  
- File handling + progress display  
- Minimal, readable UI  
- Works offline, no backend required  

If you’ve ever collected small scripts and wished they lived in one organized place, TaskForge gives you that home.

---

🚀 What’s New in v1.0.0

- Fully modular job system  
- Automatic category + job menus  
- Clean execution shell for each job  
- File loader helper (with progress support)  
- Deterministic job loading order  
- Minimal black/purple UI theme  
- Works on desktop + mobile browsers  
- No frameworks, no build steps, no dependencies  

Drop your tools into /JOBS/, refresh, and they appear instantly.

---

📦 How to Add Your Own Tools

1. Create a new .js file inside /JOBS/  
2. Give it a menuName and optional category  
3. Write whatever UI + logic you want inside app.innerHTML  
4. TaskForge handles the rest  

No configuration.  
No boilerplate.  
No build system.
